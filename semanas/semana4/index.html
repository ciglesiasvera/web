Semana 4: Box Model y JavaScript Básico

SESIÓN 1: El Modelo de Cajas (Box Model)

El Box Model define cómo se calculan las dimensiones de los elementos en CSS.

Propiedades Clave:

  • width y height: Definen el tamaño del contenido del elemento.
  • padding: Espacio entre el contenido y el borde del elemento.
  • border: Borde que rodea el padding (si existe).
  • margin: Espacio exterior al borde, que separa el elemento de otros elementos.

Diferencias entre content-box y border-box:

  • content-box: El ancho y alto especificados solo incluyen el contenido. El padding y el borde se añaden alrededor del contenido.
  • border-box: El ancho y alto especificados incluyen el contenido, el padding y el borde.

Content-Box

Content-box es como una caja donde solo cuentas el espacio interior donde pones tus cosas.

Por ejemplo, si quieres una caja que mida 20 cm de ancho y 20 cm de alto para tus cosas, con content-box, esos 20 cm solo miden el espacio interior donde van las cosas. Si añades 5 cm de relleno y un borde de 2 cm, tu caja total medirá 20 cm + 5 cm + 2 cm = 27 cm de ancho y alto.

Border-Box

Border-box es como una caja donde cuentas todo: el espacio interior para tus cosas, el relleno y el borde.

Por ejemplo, si quieres una caja que mida 20 cm de ancho y 20 cm de alto para tus cosas, con border-box, esos 20 cm incluyen todo: el espacio para las cosas, el relleno y el borde. Entonces, si añades 5 cm de relleno y un borde de 2 cm, esos se restan del espacio interior para las cosas. El espacio interior para las cosas será 20 cm - 5 cm - 2 cm = 13 cm de ancho y alto.

SESIÓN 2: Introducción a JavaScript

JavaScript es esencial para agregar interactividad y dinamismo a las páginas web.

Fundamentos de la Sintaxis Básica:

  • Variables: Declaración (var, let, const) y asignación de valores.
  • Operadores: Aritméticos (+, -, *, /, %), Comparación (==, ===, !=, !==, >, <, >=, <=), Lógicos (&&, ||, !).
  • Condicionales: Uso de if, else if, else para controlar el flujo del programa.

Tipos de Datos Primitivos:

  • number: Números enteros y de punto flotante.
  • string: Cadenas de texto.
  • boolean: Valores verdadero o falso.
  • null: Valor nulo intencional.
  • undefined: Valor no asignado.

¿Qué es console.log()?

console.log() es como un pequeño altavoz que sirve para saber lo que está haciendo tu código en cada paso. Cuando programas, a veces tu código no hace lo que esperas. console.log() te ayuda a ver qué está pasando dentro de tu programa, paso a paso.

// Ejemplo de console.log()
let numero1 = 10;
let numero2 = 4;
let suma = numero1 + numero2;

// Usamos console.log() para ver el resultado
console.log("La suma de " + numero1 + " y " + numero2 + " es: " + suma);
