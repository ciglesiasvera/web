Verificar si un Número es Par o Impar

Este ejercicio utiliza JavaScript para determinar si un número es par o impar.

¿Cómo funciona?

Un número es par si al dividirlo por 2, el resto (o módulo) es 0. Si el resto es 1, el número es impar.

Código JavaScript

function verificarParImpar() {
    // Obtener el valor ingresado
    const numero = parseInt(document.getElementById("numero").value);
    const resultado = document.getElementById("resultado");
    
    // Verificar si es un número válido
    if (isNaN(numero)) {
        resultado.textContent = "Por favor, ingresa un número válido.";
        return;
    }
    
    // Verificar si es par o impar
    if (numero % 2 === 0) {
        resultado.textContent = numero + " es un número PAR.";
    } else {
        resultado.textContent = numero + " es un número IMPAR.";
    }
}
