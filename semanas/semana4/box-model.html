Ejemplo del Box Model en CSS

A continuación se muestran dos ejemplos del Box Model con diferentes configuraciones:

Box Model con content-box
Box Model con border-box

Explicación

Ambas cajas tienen las mismas propiedades declaradas:

  • width: 200px
  • padding: 20px
  • border: 10px
  • margin: 20px

Sin embargo, la diferencia está en cómo se calcula el tamaño final:

  • content-box: El ancho total es 200px (contenido) + 40px (padding) + 20px (border) = 260px
  • border-box: El ancho total es 200px, que incluye contenido (140px) + padding (40px) + border (20px)
