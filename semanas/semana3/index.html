Semana 2: Introducción a CSS

Objetivos de la clase:

  • Comprender qué es CSS.
  • Aprender cómo se integra CSS en HTML.
  • Conocer los tres métodos principales para aplicar estilos.
  • Aplicar estilos básicos: colores, fuentes, márgenes.

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje que describe cómo se debe mostrar un documento HTML. Permite separar el contenido de la presentación visual.

Métodos para aplicar CSS

  • En línea (inline): Se escribe directamente dentro de una etiqueta HTML.
  • Interno: Se incluye dentro de la etiqueta <style> en el <head>.
  • Externo: Se enlaza mediante <link rel="stylesheet" href="estilos.css">.

Ejemplo CSS en línea

<p style="color: blue;">Hola Mundo</p>

Ejemplo CSS interno

<style>
  p {
    color: red;
  }
</style>

Propiedades básicas de CSS

  • color: color del texto.
  • background-color: color de fondo.
  • font-size: tamaño de letra.
  • margin: espacio exterior.
  • padding: espacio interior.
  • border: bordes del elemento.

Actividad práctica:

  • Crear una página HTML y aplicar estilos de las tres formas.
  • Modificar el color de fondo y tamaño de texto.
