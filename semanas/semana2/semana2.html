Objetivos de la Clase

  • Comprender la estructura básica de un documento HTML.
  • Identificar y utilizar etiquetas, elementos y atributos.
  • Entender la importancia del <!DOCTYPE>, <head> y <body>.
  • Introducir las etiquetas semánticas de HTML5 y su uso.

Parte 1: Estructura Básica

1. ¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura del contenido.

2. Estructura Básica

  • DOCTYPE: <!DOCTYPE html> define la versión de HTML.
  • <html>: raíz del documento HTML. Ejemplo: <html lang="es">
  • <head>: contiene metadatos. 
    <head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Título</title>
</head>
  • <body>: contiene el contenido visible. 
    <body>
  <h1>Bienvenidos a mi Página</h1>
  <p>Este es un párrafo.</p>
</body>

Parte 2: Etiquetas, Elementos y Atributos

  • Etiqueta: <p>Texto</p>
  • Elemento: <a href="#">Enlace</a>
  • Atributo: <img src="img.jpg" alt="Descripción">

Parte 3: Etiquetas Semánticas de HTML5

1. ¿Por qué son importantes?

  • Semántica: Ayuda a entender mejor el contenido.
  • Accesibilidad: Mejora la experiencia del usuario.

2. Principales etiquetas semánticas

  • <header>: Encabezado principal.
  • <nav>: Navegación.
  • <article>: Contenido autónomo.
  • <aside>: Contenido relacionado.
  • <footer>: Pie de página.
Ejemplo: 
<article>
  <h2>Título</h2>
  <p>Contenido del artículo.</p>
</article>

Más Conceptos Clave de HTML y Buenas Prácticas

Tipos de Etiquetas en HTML

  • Estructurales: <html>, <head>, <body>.
  • De contenido: <p>, <h1>...<h6>, <div>, <span>.
  • De medios: <img>, <audio>, <video>.
  • De formularios: <form>, <input>, <textarea>, <select>.
  • Semánticas: <header>, <nav>, <main>, <section>, <article>, <footer>.

Validación de HTML

Para asegurar que nuestro código esté correctamente escrito, es recomendable usar herramientas como el W3C Validator. Esto ayuda a prevenir errores de estructura y mejora la compatibilidad entre navegadores.

Estilos CSS: En línea, Internos y Externos

  • Estilo en línea: Dentro de una etiqueta. Ej: <p style="color:red;">Texto</p>
  • Estilo interno: Dentro de <style> en el <head>.
  • Estilo externo: En un archivo .css vinculado con <link rel="stylesheet" href="estilo.css">

Buenas Prácticas de Accesibilidad y Semántica

  • Usar etiquetas semánticas mejora la accesibilidad para personas con discapacidad.
  • Incluir atributos alt en imágenes para usuarios con lectores de pantalla.
  • Mantener una estructura jerárquica clara con encabezados <h1> a <h6>.
  • Usar <label> en formularios para asociar correctamente texto con campos.
