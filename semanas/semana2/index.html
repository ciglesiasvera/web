Objetivos de la Clase

  • Comprender la estructura básica de un documento HTML.
  • Identificar y utilizar etiquetas, elementos y atributos.
  • Entender la importancia del <!DOCTYPE>, <head> y <body>.
  • Introducir las etiquetas semánticas de HTML5 y su uso.

Parte 1: Estructura Básica

1. ¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura del contenido.

2. Estructura Básica

  • DOCTYPE: <!DOCTYPE html> define la versión de HTML.
  • <html>: raíz del documento HTML. Ejemplo: <html lang="es">
  • <head>: contiene metadatos. 
    <head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Título</title>
</head>
  • <body>: contiene el contenido visible. 
    <body>
  <h1>Bienvenidos a mi Página</h1>
  <p>Este es un párrafo.</p>
</body>

Parte 2: Etiquetas, Elementos y Atributos

  • Etiqueta: <p>Texto</p>
  • Elemento: <a href="#">Enlace</a>
  • Atributo: <img src="img.jpg" alt="Descripción">

Parte 3: Etiquetas Semánticas de HTML5

1. ¿Por qué son importantes?

  • Semántica: Ayuda a entender mejor el contenido.
  • Accesibilidad: Mejora la experiencia del usuario.

2. Principales etiquetas semánticas

  • <header>: Encabezado principal.
  • <nav>: Navegación.
  • <article>: Contenido autónomo.
  • <aside>: Contenido relacionado.
  • <footer>: Pie de página.
Ejemplo: 
<article>
  <h2>Título</h2>
  <p>Contenido del artículo.</p>
</article>

Actividades Prácticas

  • Ejercicio 1: Crear un documento HTML con las etiquetas <html>, <head> y <body>.
  • Ejercicio 2: Usar etiquetas semánticas en un documento HTML.
