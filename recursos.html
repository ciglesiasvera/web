Recursos y Herramientas

Herramientas Esenciales para el Curso

A continuación encontrarás las herramientas que utilizaremos durante el curso de Desarrollo y Diseño Web + IA. Cada una de estas herramientas es fundamental para el desarrollo de los proyectos y actividades prácticas.

Visual Studio Code

Editor de código fuente desarrollado por Microsoft. Incluye soporte para depuración, control de Git integrado, resaltado de sintaxis, y extensiones para múltiples lenguajes.

Descargar VS Code

Figma

Herramienta de diseño de interfaces basada en la web. Permite crear prototipos, wireframes y diseños colaborativos en tiempo real.

Ir a Figma

GitHub

Plataforma de desarrollo colaborativo para alojar proyectos utilizando el sistema de control de versiones Git. Esencial para el trabajo en equipo.

Ir a GitHub

XAMPP

Distribución de Apache que contiene MariaDB, PHP y Perl. Permite instalar de forma sencilla un servidor local para desarrollo web.

Descargar XAMPP

WAMP

Entorno de desarrollo web para Windows que incluye Apache, MySQL y PHP. Alternativa a XAMPP para sistemas Windows.

Descargar WAMP

W3C Validator

Servicio de validación de código HTML y CSS del World Wide Web Consortium. Verifica que tu código cumpla con los estándares web.

Ir a W3C Validator

W3Schools

Sitio web educativo con tutoriales y referencias sobre lenguajes de desarrollo web como HTML, CSS, JavaScript, PHP, SQL y más.

Ir a W3Schools

Bootstrap

Framework CSS y JavaScript para diseño de sitios web responsivos y móviles. Facilita la creación de interfaces modernas y consistentes.

Ir a Bootstrap

Extensiones Recomendadas para VS Code

  • Live Server: Lanza un servidor local con recarga en vivo para páginas estáticas y dinámicas.
  • HTML CSS Support: Proporciona autocompletado para HTML y CSS.
  • Prettier: Formateador de código para mantener un estilo consistente.
  • ESLint: Integra ESLint en VS Code para detectar problemas en el código JavaScript.
  • PHP Intelephense: Proporciona autocompletado y análisis de código PHP.
  • GitLens: Mejora las capacidades de Git en VS Code.

Recursos Adicionales

  • MDN Web Docs: Documentación completa sobre tecnologías web.
  • CSS-Tricks: Tutoriales y trucos para CSS.
  • Manual de PHP: Documentación oficial de PHP.
  • Documentación MySQL: Referencia completa de MySQL.
